1) Install python 3.
2) Install npm.
3) Install yarn.
4) Install node.js "https://nodejs.org/en/download/".
5) Open command line, change directory to project (wall segmentation) folder , then type "pip install  -r requirements.txt".
6) Run App.py file on any IDE.
7) Open VScode on wall 'segmentation\emirate-paints' directory and type "npm start" on the terminal.
8) The server should start working now.
